Troubleshooting

Common issues and solutions

üîß Need Help?

If you can't find a solution below, contact our support team at DGC2MHNE@proton.me

Login Issues

‚ùå MetaMask Not Connecting

The "Connect MetaMask" button doesn't respond or shows an error.

‚úÖ Solution:
  • Make sure MetaMask extension is installed in your browser
  • Check that MetaMask is unlocked (enter your password)
  • Try refreshing the page
  • Disable other wallet extensions that might conflict
‚ùå Wrong Network Error

You see a message about being on the wrong network.

‚úÖ Solution:
  • Open MetaMask and switch to the Polygon network
  • If Polygon is not available, add it manually:
    • Network Name: Polygon Mainnet
    • RPC URL: https://polygon-rpc.com
    • Chain ID: 137
    • Symbol: MATIC
‚ùå Email Login Fails

Unable to log in with email and password.

‚úÖ Solution:
  • Double-check your email address for typos
  • Ensure your password is correct (case-sensitive)
  • Try resetting your password if you forgot it
  • Clear browser cache and cookies, then try again

Upload Issues

‚ùå File Too Large

Error message says the file exceeds the maximum size.

‚úÖ Solution:
  • Maximum file size is 100MB per file
  • Compress large files before uploading
  • For videos, consider reducing resolution or using compression tools
  • Split large archives into smaller parts
‚ùå Unsupported File Format

The system doesn't accept your file type.

‚úÖ Solution:
  • Supported formats:
    • Images: JPEG, PNG, GIF
    • Videos: MP4, AVI, MOV
    • Audio: MP3, WAV, M4A
    • Documents: PDF, DOC, DOCX, TXT
    • Archives: ZIP, RAR
  • Convert your file to a supported format before uploading
‚ùå Upload Fails / Times Out

The upload process stops or shows a timeout error.

‚úÖ Solution:
  • Check your internet connection stability
  • Try uploading during off-peak hours
  • For large files, use a wired connection instead of WiFi
  • Refresh the page and try again

Access Issues

‚ùå Permission Denied

You see a message that you don't have permission to access this feature.

‚úÖ Solution:
  • Your current role may not have access to this feature
  • Check the User Roles page to understand permissions
  • Contact your administrator to update your role if needed
‚ùå Page Not Loading

Pages are blank or stuck loading.

‚úÖ Solution:
  • Clear your browser cache and cookies
  • Try using a different browser (Chrome, Firefox, Edge)
  • Disable browser extensions that might interfere
  • Check if the system is under maintenance

System Errors

‚ùå 500 Internal Server Error

The server encountered an unexpected error.

‚úÖ Solution:
  • Wait a few minutes and try again
  • The system may be experiencing high load
  • If the problem persists, contact the administrator
‚ùå Network Connection Error

Unable to connect to the server.

‚úÖ Solution:
  • Check your internet connection
  • Try refreshing the page
  • Check if you can access other websites
  • The server may be temporarily offline for maintenance
‚ùå Database Error

Error messages related to database operations.

‚úÖ Solution:
  • The system may be under maintenance
  • Wait a few minutes and try again
  • Contact the system administrator if the error persists
